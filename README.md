# WhatsApp AI Ordering Bot

Built using:
- n8n
- Google Gemini API
- WhatsApp Cloud API
- Google Sheets (Inventory + Orders)
- AI Agent with Memory

Features:
- Auto reply using AI
- Inventory check
- Order confirmation
- Google Sheets integration
- Role-based structured system prompt

Architecture:
WhatsApp → n8n Trigger → AI Agent → Google Sheets → WhatsApp Response
